Etiquetas extras

Etiqueta pre

La etiqueta pre sirve para respetar espacios y líneas en blanco, al contrario de la etiqueta párrafo

Holi soy majo tengo 16 años 

Hola soy
        majo   tengo 16 años

Etiqueta abr

La etiqueta abr se utiliza para las abreviaturas

Según datos de la OMS el consumo excesivo de azúcar provoca daños severos en la salud de los niños

Etiqueta bdo

Si desea cambiar la direción de un texto, se utiliza la etiqueta bdo

Este es un texto normal

Este es un texto diferente

Etiqueta cite

La etiqueta cite se utiliza para escribir citas textuales

If you love me don't let go por: Majo

Etiqueta small

La etiqueta small hace el texto mas pequeño

Este texto está dentro de una etiqueta samll

Etiqueta sub

Si se desea escribir un subíndice, se debe utilizar la etiqueta sub

1503

Etiqueta sup

Si se desea escribir un superíndice, se debe utilizar la etiqueta sup

1503

Etiqueta time

Para escribir fechas y horas se debe utilizar la etiqueta time

Mañana es día 2024-11-06

Mañana es día 6 de noviembre

Etiqueta adress

Para especificar al navegador que el texto se refiere a una dirección utiliza la etiqueta adress

Loma dorada No.100 Colonia Lomas de Rosales CP 89100 Tampico,Tams
